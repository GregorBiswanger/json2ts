# json2ts

json2ts converts a JSON to TypeScript interfaces.

## Installation

```
npm install json2ts
```

## Usage
json2ts can be used as a Node.js module:

```
let json2ts = require("json2ts");
let result = json2ts.convert(jsonContent);

```
